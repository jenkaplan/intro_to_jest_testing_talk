Why Jest?

  • Most Mocha testing for React is from 2015 or early 2016
  • Jest is built by Facebook
  • Was neglected for a long time, but for the past year they completely rewrote it and now it is great
  • Downside is there is not many docs with complex examples yet

What is Snapshot Testing?

  • Allows you to quickly write testing for static data
  • If the data changes, you don't need to rewrite your test, it will do it by running `-u` after your test command
